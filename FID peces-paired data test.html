Codigo utilizado para el TFG títulado "El comportamiento de huida se ve afectado por el tipo de presión antropogénica: el caso de la lubina y la salpa"¶

Contexto: Las distancias a las que los pescadores pueden acercarse a los peces (FID: Flight Innitial Distance) debera ser diferente de acuerdo a la predacin humana que estos sufren, si esto fuera cierto, se esperara que la lubina tuviera distancias ms grandes. Utilizamos una muestra de 23 pescadores para analizar estas diferencias en la percepcin de la distancias entre las dos especies. Para hacer esta comparacin vamos a usar una prueba de comparacin de medias parmetrica de medidas dependientes (los datos vienen emparejados: dos observaciones de FID por cada pescador) si los datos cumplen con los supuestos de la prueba (Student t-test), o si no se cumplen, una prueba no paramtrica (Mann Whitney W test). Tambin nos interesa demontrar si hay diferencias en la proporcin de pescadores que prefieren pescar peces pequeos frente a peces grandes, debido al tamao de muestra y la frecuencia tan baja de pescadores que prefieren pequeos (n = 1) se recomienda usar una prueba exacta binomial de bondad de ajuste.

Peces grandes o peces pequeños¶

La prueba exacta de bondad de ajuste se utiliza para comparar las proporciones de espacio muestral de una variable categrica nominal con una proporcin esperada, en nuestro caso la variable es dicotmica ya que puede tomar nicamente dos valores: preferencia por peces pequeos o grandes. La prueba se denomina exacta ya que no calculamos un estadstico de prueba que mida qu tan diferentes son los valores observados de los esperados mediante un estadstico como el chi cuadrado, sino que directamente calculamos la probabilidad de obtener la proporcin esperada bajo la hiptesis nula (en nuestro caso la probabilidad de que una persona de 22 prefiera peces pequeos en base a que es esperara 50%). Se utiliz una prueba binomial y no un Chi Cuadrado ya que el nmero de pescadores en una celda es solo uno, lo que no se reocmienda para un chi cuadrado.

Ms explicaciones: https://rpubs.com/juanhklopper/exact_test_of_goodness_of_fit

#Binomial exact goodness of fit test
n <- 21
k <- 1
p <- 0.5

x <- seq(0, n)
y <- dbinom(x,
            size = n,
            prob = p)

barplot(height = y,
        names.arg = x,
        col = "deepskyblue",
        main = "Probabilidad de preferir peces pequeos",
        xlab = "Numero de personas",
